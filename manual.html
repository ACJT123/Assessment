1. Clone the repository
  2. Go to 'employee-backend' by entering 'cd employee-backend' and run 'npm install', then run 'npm start' to start the server
  3. Open another terminal and Go to 'employee-frontend' by entering 'cd employee-frontend' and run 'npm install', then run 'npm run dev' to start the application
  4. Open your browser and go to 'http://localhost:5173' to view the application
